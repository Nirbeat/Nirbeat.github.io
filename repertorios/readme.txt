La app presentada toma datos de un JSON donde figura el repertorio de una banda con las duraciones de cada tema, y crea un formulario con elementos checkbox.

Se eligen los temas que potencialmente podrían tocarse y el boton "Calcular" te notifica del tiempo total del repertorio.

La app se puede adaptar a cualquier numero de temas en el repertorio total y siempre generará un checkbox por canción. Eso significa que es adaptable a cualquier base
de datos, y aprovechando eso, la usaré para las bandas en las que participo. Actualmente figuran dos bandas en un input de tipo select, donde se puede comprobar
que la app funciona para cualquier recogida de datos.

El próximo paso es, efectivamente, que consulte una tabla SQL en lugar de un JSON, pero aún no domino lo suficiente lenguajes de backend.
