Para ser honesto, este proyecto lo tengo descuidado hace ya tiempo.

La idea era hacer una app de escritorio para la gestoría de seguros de mis hermanos, diseñando base de datos, sistema de cobranza, programando notificaciones
de vencimientos, resumenes de facturación, etcétera.

En lo último que trabajé, había logrado que se guardara un objeto con los datos del asegurado y se almacenara en localstorage. La idea a futuro es que los datos
se guarden en una tabla SQL para consulta: en caso de renovación del seguro, por ejemplo, se buscaría por DNI al cliente y se podría expedir un nuev comprobante
de pago sin tener que repetir el ingreso de datos. También serviría para agilizar notificaciones de vencimiento.

Este proyecto data de mis primeros meses codeando (fue mi primer idea para practicar, de hecho) y estoy muy orgulloso de haber podido llegar a eso en tan poco
tiempo.
